summary: |
  This is my highest recommended framework for JavaScript/TypeScript based projects. It offers top tier performance, is easy to set up and offers an opinionated project structure that makes projects well structured by default. Combined with the fact Vue components are kept in separate `.vue` files keeping them small you're even encouraged to not fall into the pitfalls of spaghetti code that frameworks like React that are less opinionated let you fall into.

  This section only goes over the bare minimum for Nuxt, you can [read more about Vue in its section.](/frontend/vue)
